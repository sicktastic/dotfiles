##### Installation

Clone repo to your desired directory:

```bash
git clone git@github.com:antwonlee/dotfiles.git
```

Link your .vimrc accordingly:

```bash
ln -s ~/Site/dotfiles/vim/vimrc ~/.vimrc
```
